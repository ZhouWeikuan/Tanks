void Object::update ( Uint32  dt)
virtual

Uaktualnienie prostokątan dest_rect na podstawie pozycji obiektu: pos_x, pos_y. Odliczanie czasu wyświetlania jednej klatki animacji i zmiana klatki po odliczeniu opowiedniego czasu.

Parametry
dt- czas od ostatniego wywołania funkcji, wykorzystywany do odliczania czasu wyświetlania klatki

Reimplementowana w Player, Tank, Bonus, Enemy, Brick, Bullet i Eagle.
