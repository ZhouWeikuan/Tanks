void Player::changeStarCountBy ( int  c)

Funkcja zmienia liczbę aktualnie posiadanych gwiazdek. Przy niezerowej liczbie gwiazdek zwiększana jest domyślna prędkość czołgu, a dla liczby gwiazdek większej od 1 oraz dla każdego dodatniego c zwiększana jest maksymalna liczba pocisków.

Parametry
c- zmiana liczby gwiazdek, może być ujemna
